What kind of JSON Objects or Data Types does this support ?

This tool supports JSON Object, JSON Array, String, Number, Boolean & Null types

How to make a key optional ?

if RAML has sampleKey: string and you want to make this optional then change it to sampleKey?: string

How to add another data type for a key ?

if RAML has sampleKey: string and you want to make this accept both string or null then change it to sampleKey: string | nil
